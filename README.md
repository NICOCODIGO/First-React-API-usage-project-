# [React IP lookup App](https://first-react-api-usage-project-g218.vercel.app/)

Hello! In this project of mine, I have built this web to continue learning and understanding JavaScript, CSS & HTML. It has two parts:

No server or backend — just a browser-only project.
This is my first personal project working with the React framework. I kept it as simple as I could to help me learn it, and to use an external API source to also make me understand how data is used and processed

<h3>💡 What it is: </h3>
A single-page React web app built with Vite that uses the IPQuery API to fetch geolocation and network data for any IP address input (IPv4, IPv6)

# 🚀 Features
- Displays your own **public** IP on page load (won't update in real time when switching to vpn in the middle of it)
- Lets you search any IPv4 or IPv6 address
- Shows location, ISP, and VPN/proxy detection (If applicable)
- Validates input and shows "Invalid input" for bad entries 
- Modular structure with separate folders for API, components, styles, and utilities

# 🧰 Techs:
- NPM 
- React (Vite)
- JavaScript (ES6)
- IPQuery.io API
